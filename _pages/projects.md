## Tracing L1 Interference in English Learner Writing

**EMNLP 2025**

A longitudinal learner corpus with error annotations designed to support research on native-language transfer in learner English. The project provides infrastructure for studying how interference patterns emerge over time and for evaluating whether language models can reproduce them as simulated learners.

## Controlled Generation for Language Pedagogy

**ETH Zurich**

Developed a controllable generation direction for pedagogical text generation, representing lesson objectives as explicit constraints such as target constructions, proficiency level, and vocabulary limits. Also contributed to an SNSF proposal operationalizing this agenda through constrained decoding (sequential Monte Carlo), an instructor-facing constraint specification workflow, and classroom-style evaluation.

## Structure-Aware Evaluation for Chinese Character Modeling

**ETH Zurich**

Proposed a structure-aware evaluation suite aimed at testing sub-character knowledge rather than relying only on word-level semantics, with tasks and metrics for more structurally informed assessment.

## Low-Resource and Interpretable Machine Translation for Indian Languages

**National Language Translation Mission**

Worked on low-resource NMT strategies and an interpretable machine translation framework for Hindi, Kannada, and Sanskrit using a Paninian-based interlingua. Built a joint Hindi-Kannada neural dependency parser using Trankit and introduced stronger engineering practices through code review, documentation, and testing.

## Multiscript Dictionary OCR

**Independent project**

Designed a multilingual OCR web application for recognizing text in images and supporting dictionary search across scripts, with applications for language learners and linguists.

## Large-Scale Data Systems and Infrastructure

**Industry**

Built and maintained real-time and high-volume data systems across industrial manufacturing and healthcare contexts, including ETL pipelines, telemetry, distributed tracing, APIs, and production infrastructure. This work informs my applied understanding of scalable machine learning systems.
